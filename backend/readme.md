A `.env` file is required with the following required variables:
```
POSTGRES_DB=
POSTGRES_PASSWORD=
POSTGRES_USER=
POSTGRES_PORT= [optional, default is 5432]
AUTH0_DOMAIN=<auth0 application domain>
AUTH0_AUDIENCE=<auth0 API audience>
CORS_ALLOWED_ORIGINS=<frontend app url>,<app url 2 etc.>
EXTERNAL_API_URL=https://example.com
EXTERNAL_API_KEY=
EXTERNAL_API_HOST=
```
